# 🎯 Руководство по улучшению распознавания лиц FaceGuard

## 📊 Текущая конфигурация
- **Модель**: LBPH (быстрая, локальная)
- **Порог**: 70 (средний)
- **Разрешение камеры**: 1280x720

## 🔧 Способы улучшения распознавания

### 1. 📸 Качество обучающих данных (САМОЕ ВАЖНОЕ!)

**Требования к фотографиям:**
- ✅ Минимум **8-15 фотографий** на каждого человека
- ✅ Разные ракурсы: фронт, левый профиль, правый профиль, 3/4
- ✅ Разное освещение: яркое, тусклое, боковое
- ✅ С очками и без (если носит)
- ✅ Разные выражения лица
- ✅ Хорошее качество (не размытые, не слишком тёмные)
- ✅ Лицо занимает минимум 30% кадра
- ❌ Избегать: размытые фото, слишком тёмные, закрытое лицо

**Как добавить фотографии:**
1. Зайти в веб-интерфейс FaceGuard
2. Перейти к профилю человека
3. Загрузить 8-15 качественных фотографий
4. После загрузки — переобучить модель (кнопка "Retrain Model")

---

### 2. 🤖 Выбор модели распознавания

#### Вариант A: LBPH (по умолчанию)
**Плюсы:**
- ⚡ Очень быстрая
- 💻 Работает на слабом железе
- 📦 Не требует дополнительных библиотек

**Минусы:**
- 📉 Менее точная
- 🌡️ Чувствительна к освещению

**Настройка в `.env`:**
```env
RECOGNITION_MODEL=lbph
RECOGNITION_THRESHOLD=60  # 50-60: строго, 60-70: сбалансированно, 70-90: мягко
```

#### Вариант B: DeepFace (рекомендуется для точности)
**Плюсы:**
- 🎯 Гораздо точнее
- 🌟 Устойчива к изменениям освещения
- 🧠 Использует глубокое обучение

**Минусы:**
- 🐌 Медленнее LBPH
- 💪 Требует больше CPU/RAM
- 📥 Первый запуск скачает модели (~100MB)

**Настройка в `.env`:**
```env
RECOGNITION_MODEL=deepface
DEEPFACE_MODEL=Facenet512  # Рекомендуется
DEEPFACE_DISTANCE_METRIC=cosine
```

**Доступные модели DeepFace (от самых точных):**
1. **ArcFace** — 🥇 Максимальная точность, но медленная
2. **Facenet512** — 🥈 Отличный баланс скорости и точности (рекомендуется)
3. **VGG-Face** — 🥉 Хорошая точность
4. **Facenet** — Быстрее Facenet512, чуть менее точная
5. **Dlib** — Хорошая, но медленная
6. **OpenFace** — Легкая, но менее точная

---

### 3. ⚙️ Настройка параметров детекции

**Параметры в `.env`:**

```env
# Минимальный размер лица в пикселях
MIN_FACE_SIZE=100
# Рекомендации: 80-100 для близких лиц, 60-80 для дальних

# Точность детекции (меньше = точнее, но медленнее)
FACE_SCALE_FACTOR=1.1
# Рекомендации: 1.05-1.1 точно, 1.2-1.3 быстро

# Минимум соседних детекций (больше = меньше ложных срабатываний)
FACE_MIN_NEIGHBORS=6
# Рекомендации: 5-7 для баланса, 3-4 для чувствительности
```

---

### 4. 🎥 Настройка камеры

```env
CAMERA_WIDTH=1280
CAMERA_HEIGHT=720
CAMERA_FPS=30
```

**Рекомендации:**
- Для распознавания достаточно 720p (1280x720)
- Не ставить слишком высокий FPS (30 достаточно)
- Убедитесь в хорошем освещении в месте установки камеры

## 🚀 Пошаговая инструкция по оптимизации

### Шаг 1: Улучшите качество данных
1. Для каждого человека загрузите 8-15 качественных фотографий
2. Убедитесь в разнообразии ракурсов и освещения
3. Переобучите модель

### Шаг 2: Выберите конфигурацию

**Если у вас слабое железо или нужна скорость:**
```env
RECOGNITION_MODEL=lbph
RECOGNITION_THRESHOLD=60
MIN_FACE_SIZE=100
FACE_SCALE_FACTOR=1.1
FACE_MIN_NEIGHBORS=6
```

**Если нужна максимальная точность:**
```env
RECOGNITION_MODEL=deepface
DEEPFACE_MODEL=Facenet512
DEEPFACE_DISTANCE_METRIC=cosine
MIN_FACE_SIZE=100
FACE_SCALE_FACTOR=1.1
FACE_MIN_NEIGHBORS=6
```

### Шаг 3: Примените настройки
1. Скопируйте выбранную конфигурацию в `agent/.env`
2. Перезапустите агент:
   ```bash
   cd agent
   python main.py
   ```

### Шаг 4: Тестирование
1. Проверьте распознавание в разных условиях освещения
2. Проверьте с разных расстояний
3. Проверьте с очками/без
4. Если есть ложные срабатывания — уменьшите `RECOGNITION_THRESHOLD`
5. Если не распознаёт — увеличьте `RECOGNITION_THRESHOLD`

## 🔍 Диагностика проблем

### Проблема: Не распознаёт знакомых людей
**Решения:**
1. ✅ Добавьте больше фотографий (минимум 8-15)
2. ✅ Увеличьте `RECOGNITION_THRESHOLD` (например, до 80)
3. ✅ Переключитесь на DeepFace
4. ✅ Проверьте освещение

### Проблема: Распознаёт незнакомых как знакомых
**Решения:**
1. ✅ Уменьшите `RECOGNITION_THRESHOLD` (например, до 50-60)
2. ✅ Увеличьте `FACE_MIN_NEIGHBORS` до 7
3. ✅ Увеличьте `MIN_FACE_SIZE` до 120

### Проблема: Слишком медленно
**Решения:**
1. ✅ Используйте LBPH вместо DeepFace
2. ✅ Увеличьте `FACE_SCALE_FACTOR` до 1.3
3. ✅ Уменьшите разрешение камеры до 640x480

### Проблема: Не детектирует лица
**Решения:**
1. ✅ Уменьшите `MIN_FACE_SIZE` до 60
2. ✅ Уменьшите `FACE_MIN_NEIGHBORS` до 4
3. ✅ Проверьте освещение и положение камеры
